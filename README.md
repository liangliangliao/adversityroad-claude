# 逆境之路 AdversityRoad

Unity 第三人称动作冒险 RPG：把现实中的困境转译为可战斗、可识别、可反击、可成长的敌人与场景。

> 看见弱点 → 面对困境 → 主动战斗 → 失败复盘 → 强化意志 → 再次挑战 → 逆袭成长

- 引擎：Unity 6000.5（URP，安卓/PC 双端）
- 打包：GitHub Actions + game-ci/unity-builder（推送到 `main` 或 `claude/**` 分支自动构建安卓 APK，Library 已缓存）

### 方案对齐 · 四大新关卡（本次迭代）

按最终商业化方案把 Demo 四关一 Boss 补齐成型：

- **小题大做审判庭（方案关卡一）**：倾斜的法官席 + 两侧旁观者席阴影；6 枚**浮动标签**

  **证据桌**走近即"看清事实"——此后标签一击即碎（事实先于评价）；**破碎镜子**回补自尊。
  Boss【自我否定法槌】三阶段：**标签弹幕 → 审判冲击波（大红圈预警）→ 自我否定重锤（追身落锤）**
- **噪声放大（方案关卡二 Boss）**：回到噪声街区，街心广场上的【刺激放大器】——
  周期性**噪声放大**（全场专注冲击，定心格挡/不读心盾可整个化解）；制造**幻影假目标**
  （攻击幻影=把力气花在猜测上：掉专注、涨反刍）——「不读心盾」使幻影显形消散、
  「注意力回收」全场清除后猛攻真身
- **拖延沼泽（方案关卡三）**：浅泥轻减速/深泥重减速+行动力流失、**干地平台**落脚、
  **手机光点区**（站入即流失专注与行动力）、**目标石板**（站上恢复行动力）、床铺藤蔓与漂浮计划纸。
  Boss【明天之王】**泥壳护体**（伤害大减打不动）——点燃场边**三座五分钟火种台**（走近即点燃）
  泥壳崩裂进入长破绽；战斗中召唤明日泥怪、往玩家脚下浇灌临时深泥
- **旧事回声馆（方案 Demo Boss·旧我回声）**：博物馆展柜长廊——靠近展柜触发**旧事回声**（反刍上升），
  站定完成**归档**（看见事实→命名感受→提取边界→转化行动），归档 3 座开启终局大门；
  失败展廊有旧话复读者/过去判官/反刍虫群把守。终局 Boss【旧我】四阶段：
  **旧话复读 → 身份冻结（「五分钟火种」解冻）→ 失败召回（召唤前章旧敌幻影）→ 整合选择**——
  残血后旧我停手且无法再被攻击（血线保护），走进**整合圆环**完成「旧我整合式」：
  **旧我化为影子护卫**随行（永久保存，靠近时反刍消退加速）。
  核心台词：「你曾经保护过我，但现在我要继续往前。」

### 战斗系统（借鉴黑神话悟空框架）

- **双系连段（招式全部绑定 Mixamo 动作库真实动作）**：
  拳=巨剑连斩「巨剑横斩→巨剑撩斩→突刺→巨剑旋风斩」，腿=腿法连段「正踢→侧踹→旋身空翻踢→飞踢」，
  两系可自由混接；动画从发力相位起播+提速播放，命中后即开取消窗口——连点无缝衔接不糊招
- **组合招式（连点拳/腿按序自动成招）**：拳拳拳=三连斩·大回旋、腿腿腿=三连腿·裂空飞踢、
  拳拳腿=双斩·空翻旋踢、腿腿拳=双腿·贯心突刺；2 势绝招：拳拳腿腿=龙卷·旋风绝斩、
  腿腿拳拳=踏空·裂地跳劈、拳腿拳腿=剑影·惊鸿飞踢（终结动作均取动作库大招）
- **摇杆磁吸锁敌**：攻击范围内有敌人即自动咬住目标出招（摇杆用于多敌间选目标、或明确指空处脱离），
  磁吸突进差多远冲多远，连点每一击都实打实命中
- **意势系统**（棍势式资源，0-3 点）：命中轻击/完美闪避/蓄力积攒，HUD 圆点显示，满势脚下亮起光环
- **蓄力重击**：按住「重」沉腰举械蓄力（耗体力、积意势），松开释放**蓄力·巨剑跳劈**；
  **势≥2 触发旋风终结**（消耗全部意势大幅增伤）；连段中轻点「重」派生**切手·撩斩**快速反击；
  指令技（轻点重+方向）：前=疾影突刺、后=旋身空翻踢、左右=旋风斩；跳中按重=空袭·裂地跳劈
- **完美闪避**：在敌人攻击命中瞬间翻滚 → 短暂时缓 + 意势+1 + 下一击必暴击（敌人攻击带 0.28s 前摇可读招）
- **空中攻击**：跳+拳=空袭跳劈、跳+腿=飞踢、跳+重=空袭·裂地跳劈；
  **蹲伏攻击**：蹲+腿=扫堂腿（贴地环扫、高削韧）、蹲+拳=低位突刺（下段直线戳击）
- **每招独立攻击范围**（长/宽/高/轨迹各不相同，技能越高范围越大）：突刺长而窄（直线）、
  横斩横宽（横扫弧）、撩斩纵高、旋风斩/扫堂腿环身 360°、跳劈罩住落点；
  蓄力越满范围越大，绝招终结与技能级攻击范围最大
- **双角色 + 武器库 + 面具库（角色/武器/面具三者资产分离，内置 glTFast 支持 glb/gltf）**：
  「角色」面板先选角色（角色·贰=`PlayerModel2`，glb/gltf/fbx 均可、动作沿用角色壹动作库、
  缺失自动回退；**异源骨架自动对齐**——glb 角色骨名无 mixamorig 前缀/层级不同时自动改名
  与链路对齐，默认动作库直接可用），再从武器库（`Characters/Weapons/`，glb/fbx/gltf/zip）
  选武器——默认持剑（模型无自带兵器时程序化长剑兜底），重选即替换：**握持对齐+手指握拳**
  （复刻角色壹自带巨剑的世界空间握姿、柄端入掌心、五指绕柄轴卷曲攥住剑柄）；
  再从面具库（`Characters/Masks/`，fbx/glb/gltf/zip）选面具戴脸——自动定尺贴脸、随头骨转动，
  支持多面具切换；三者选择均本地持久化。动作库补位：放入 Blocking/Stand To Roll/Stunned/
  Great Sword Casting/Leg Sweep 等专用片段即自动替换现有替代动作（见 MIXAMO_SETUP.md）
- 格挡（边界盾）/ 精准格挡（定心格挡化解心理攻击）；锁定与互动全自动化（无需按键）
- **战斗可读性（对标商业动作游戏的产品化打磨）**：
  - **近镜角色淡出**：镜头贴近任何角色（近身缠斗/贴墙）时该角色整体淡为半透明，
    根治"整屏被模型糊脸/镜头穿模"；镜头碰撞回缩下限抬高，不再缩进身体
  - **角色保持原色**：玩家与敌人均不做整体染色，保留模型原本材质与肤色；
    敌我识别交给头顶状态条、前摇红圈与恶意台词
  - **屏幕固定连击计数器**（格斗游戏惯例）：连击数固定在屏幕右侧弹缩显示、越高越红、断连淡出，
    不再跟着命中点满场乱飞；招式名浮字节流（同名招 0.9s 内不重复弹），战斗信息不刷屏
  - **夜战保底亮度**：夜晚/暗场景环境光与主光下限上调，看清双方拳脚与前摇优先于氛围
- **部位受击反应**：按命中点定位身体部位（头/躯干/腰腹/左右腿），被打部位向打击方向甩动回弹
  （打头甩头、扫腿屈膝），接触点弹出部位标签；冲量可叠加——连打两下头就有两次可见甩头；
  重击倒地为「当场栽倒」：受击位移只在身体后仰头几帧内完成，全程零漂移滑行
- **战斗反馈**：伤害数字、受击闪红、碎块飞溅、顿帧、震屏、程序化动作姿态（前倾挥击/翻滚/倒地/僵直/施法/死亡倒伏消散）；
  **程序化音效**（零外部资产，运行时 PCM 合成）——挥击风声/命中闷响/重击/格挡/定心弹反/前摇警示/受击/死亡各有其声；
  刀光改为细窄半透明加色的一道光（非大色块），绝招/组合技为「能量光爆」（强闪光+放射光条+上升火星+细冲击环）而非平铺色板
- **招式分工**：拳=快攻（出手快、低击退、高频衔接），腿=击退（大幅推开、打断突进），重击=破防（高削韧击破霸体），
  防御=减伤（体力格挡免伤 80%），闪避=反击（完美闪避时缓+返势+必暴击），意志技/定心格挡=反制心理攻击
- **重击倒地**：高伤害攻击将玩家击倒，倒地演出 + 起身无敌帧
- **敌人头顶状态条**：名字+等级、血条、韧性条、实时情绪（窥伺/紧逼/狰狞/讥讽/慌乱）

### 心理攻击（语言+情绪层面）

- 敌人实时**恶意台词**：头顶气泡 + 底部字幕，按弱点轴与所在场景组合（追击中持续低语，心理攻击时高声喊话）
- 受击**暗角脉冲**：物理伤害红色、心理伤害紫色
- **AI 提示词配置**（游戏内「AI台词」面板）：全局提示词 + 各场景提示词均有**默认模板**，可随时修改
- **云端 LLM 台词生成**：支持 **OpenRouter / DeepSeek / EdenAI**（OpenAI 兼容 Chat 接口），
  游戏内配置 API Key 与模型；**延迟方案**——进入区域即后台预取一批台词入池（区域×弱点轴），
  敌人喊话即取即用零等待，池空/断网/未配置时无缝回退本地模板

### 防晕设计（镜头/帧率/后处理/移动）

- **帧率**：安卓默认 30 帧已解锁——目标帧率取屏幕刷新率（60–120），并启用 Frame Pacing（Swappy）稳定帧间隔
- **镜头**：跟随点为平滑 CameraTarget（水平/纵向均临界阻尼软跟随，极短时间几乎无滞后又滤掉逐帧
  移动微抖动，消除晃屏与橡皮筋游动感），转角临界阻尼平滑无过冲，FOV 固定 62° 永不变化，
  碰撞回缩快伸出慢不弹跳，震屏改为极小纵向脉冲（无随机抖动），专注低不再摇镜头（改由暗角表达）
- **三模式运镜（角色转向快 / 位置中速跟随 / 镜头旋转慢）**：
  - **探索镜头**（无敌）：位置用中速 `SmoothDamp` 跟住玩家、角度基本固定；**玩家一转向，
    镜头即刻开始平稳缓慢地转到其背后（面朝方向）**——用大平滑时间的临界阻尼弹簧，转身即跟、
    小抖动只带来极轻微慢移不晃屏；树木/建筑遮挡自动透明
  - **战斗镜头**（锁定敌人）：镜头优先看敌人，取景点偏向「玩家↔敌人中点」让**两人同框居中**、
    压低俯仰、适度拉近看清拳脚与前摇；攻击自动朝向敌人，可轻微围敌调整但不乱晃
  - **大招镜头**（仅超必杀触发）：短暂拉近 + 轻微慢动作 + 命中小震，结束自动回稳；
    **普通移动/普通攻击都不会触发这种镜头，普攻也不再频繁震屏**（防晕）
  - **第一人称**：镜头在眼睛高度、脸前方，俯仰自由——低头看见自己的手/脚/剑，抬头看见天空，
    挥剑/踢腿低头即见肢体与兵器在空中运动；只隐藏头部，身体其余可见
  - 底层：转角/位置均临界阻尼平滑无过冲
- **遮挡自动透明**：树木（含无碰撞体的树冠）挡在镜头与玩家之间时自动淡出为半透明，移开即淡回
- **移动**：加减速平滑（速度无突变），摇杆全向 360° 模拟量走跑（极小死区仅滤静止噪声）；
  人形步态叠加脊柱反向扭转/躯干侧倾/骨盆侧摆/落步点头等次要动作，消除机械感
- **后处理**：运行时强制关闭运动模糊/色差/镜头畸变/噪点，暗角强度限幅

### 言语攻防 · 反刍 · 战后复盘（本次迭代）

把方案里最核心的循环——「语言打语言、事实打赖账、边界打索取、不读心打刺激、复盘代替反刍」——落成可玩机制：

- **言语攻防（快速选择式）**：敌人发动心理攻击（凝视/低语）时，屏幕下方弹出**三选一回应**面板
  （一句「最优回击」+ 两句「无效回应」，随机打乱，5 秒倒计时，战斗不暂停用非缩放时间计时）。
  - **选对**：敌人**语塞进入破绽**（削韧，可击破触发大破绽）、对应弱点属性回补、反刍下降，**且不吃这次心理伤害**
  - **选错/沉默**：照常吃心理伤害，并**额外累积反刍**（互骂/退缩/过度辩护只是把自己拖进反刍）
  - 回应库按弱点轴给出最优句（如噪声轴「我听见了，但我不跟随」、边界轴「帮助有边界，这次你自己承担」、
    求职轴「没有回应不代表没有价值，我再来一次」），全部抽象化、服务战斗机制，绝不输出现实操控话术
  - 冷却/恢复模式/已在进行时自动回退为原来的即时落伤害
- **反刍值（方向相反的消耗条）**：越满越糟。被心理攻击命中即累积，选错回应额外增加；
  战斗外缓慢消退避免软锁；上升时紫色暗角脉冲提示
- **行动力 ActionPower（本次迭代·替代"决断"）**：拖延轴攻击/深泥/手机光点消耗它，
  **过低时移速明显下降**（拖延的具象体感）；五分钟火种/火种台/目标石板恢复
- **关系消耗值 RelationshipDrain（本次迭代·第二条反向消耗条）**：被索取/责任转嫁类攻击、
  接下虚假责任时上升；**≥70 时技能冷却 ×1.5**（被掏空的注意力与精力）；
  守住边界的正确回应与边界圈使其回落，战斗外缓慢消退
- **战后复盘（安全屋核心·「复盘」面板）**：把这一战整理成**四栏——事实 / 感受 / 边界 / 行动**
  （按当前章节主题生成，抽象化不复刻真实人物/事件）；**「归档此战」清空反刍值并回补心理属性**——
  用复盘把刺痛转成经验，旧事进入档案不再无限回放

### 五种战斗姿态（本次迭代）

把方案「五种姿态」落成战斗层的策略选择——**属性条下方一排五枚姿态**，点选或按 `Tab`/`F` 循环切换：

| 姿态 | 心法 | 克制弱点轴（心理减伤 60% + 言语契合加成） | 物理输出 |
| --- | --- | --- | --- |
| 起步 | 不要想太多，先动起来 | 拖延 / 求职焦虑 / 低信心 | ×1.06 |
| 边界 | 我守住我自己 | 边界冲突 / 公平责任 | ×1.00 |
| 定心 | 外界可以存在，但不能接管我 | 噪声干扰 | ×1.00 |
| 事实 | 发生了什么，先说清楚 | 公平责任 / 求职沉默 | ×1.08 |
| 意志 | 我害怕，但我继续向前 | 意志崩塌 / 羞耻 / 自我怀疑 / 失败恐惧 | ×1.10 |

- **把姿态切到与来袭弱点轴匹配的一档**，这次心理伤害 ×0.4（减 60%）；进攻型姿态（起步/事实/意志）另有物理输出加成
- 与言语攻防联动：正确回击时若**当前姿态正好克制这条轴**，额外回补属性、更快清空反刍（「姿态契合·加成」）
- HUD 高亮当前姿态并显示心法，切换时底部字幕提示核心句

### 责任转嫁法院（终章 · 边界与责任线）

按方案「第四章 边界与责任线 / Demo 关卡四」把整座法院落成一个**自南向北纵深推进**的关卡，
物理空间、地图结构、可交互物体、战斗机制均对齐方案：

- **地图结构（三段式推进）**：法院大门（南墙缺口）→ **文件走廊**（两侧证词走廊/陪审团阴影，
  文件山与锁链闸挡路）→ **责任天平大厅**（责任天平/责任归属台/边界圈）→ **审判席**
  （最高法官台 + 法槌，Boss 在此出现）
- **可交互 / 可破坏物体**：
  - **文件山**：用兵器（**事实之刃**）连击可击碎，露出**事实证据**拾取物（回补心理属性、
    象征"用事实回应而非争对错"）
  - **锁链闸**：横跨走廊，链未断时站入**大幅减速**；劈断任一根锁链即**「破念」**解除束缚
  - **责任天平**：走近即**判断责任归属**——天平从失衡回正，**解锁/确认「责任归还」**并回补边界
  - **边界圈**：安全区，站入缓慢恢复边界、消退「过度负责」
- **责任球机制**：Boss【全责法官】周期性把「责任球」抛向玩家，玩家要**分辨真假**——
  - **红球=虚假责任**：举**边界盾**（格挡键 / 触屏「挡」）当场挡回抛球者削其韧性；
    若不挡任其命中，会附着**「过度负责」**（减速 + 边界流失 + 反刍上升）
  - **绿球=真实责任**：**别推开**，接下它回稳意志；用盾把真实责任推开=推卸本分，掉自尊
- **三件战斗工具**（对齐方案「边界盾 / 责任归还 / 事实之刃」）：
  - **边界盾**＝格挡，当场挡回飞来的责任球
  - **责任归还**＝新技能（技能键 `3` / 触屏「还」）：清除「过度负责」减速、把所有飞来的
    虚假责任球一次性打回法官、回补边界——"把不属于我的部分准确还回去"
  - **事实之刃**＝劈开文件山取证
- **法槌落判**：Boss 近身时在玩家脚下预警红圈后**落槌 AoE**（可翻滚/走出圈外躲开），
  即方案「法槌·Boss 阶段攻击源」
- 与言语攻防联动：法官心理攻击走**边界轴**，最优回击「帮助有边界，这次你自己承担」，
  **边界姿态**正好克制（心理减伤 60% + 契合加成）；战后复盘新增法院四栏
- 入口：主线打通城市广场后，从广场**东侧大门缺口**的传送门进入；「敌人+」面板亦可随处添加
  【全责法官】挑战，个性化画像中「边界/公平责任」高分者遭遇战优先匹配全责法官

### 安全屋系统（本次迭代 · 成长闭环产品化）

右上角「安全屋」枢纽汇总五大子系统（各面板均可独立打开，数据本地持久化）：

- **复盘**：四栏（事实/感受/边界/行动）改为**可编辑输入框**——默认按当前章节生成模板
  （十章各有专属文案），也可改写成自己的话；「归档此战」= 清空反刍 + 回补属性 +
  **写入旧事档案** + **复盘点 +1**（反刍 ≥10 才发点，防刷）
- **成长（技能树）**：五条路线 × 两级共十节点——边界加固/明确拒绝、专注扩容/声音过滤、
  起步惯性/火种常燃、事实之刃锋利/成本判断、自尊锚点/旧事止响；用复盘点解锁、即时生效
  （上限扩容/承伤缩减/冷却缩减/输出增益）
- **装备（五大套装）**：边界守卫套/专注夺回套/公平复盘套/行动起步套/旧我整合套——
  一次穿一套，被动倍率对应各困境线（含方案核心台词）
- **图鉴**：18 种心魔的心理轴、**克制要点**与累计击败数
- **档案**：历史复盘四栏翻页浏览——旧事从脑内回声，变成档案里安静的一页

## 设计原则

- 游戏基于真实经历抽象为敌人原型与场景气氛，**不复刻真实人物/地点/事件**。
- 心理攻击只做成游戏机制（低语/凝视/泥潭+抽象台词），**绝不输出可复制到现实的操控话术**。
- 失败不惩罚：死亡生成复盘任务，引导策略调整后再战，章节进度保留。
- 隐私数据最小化：个人材料与提示词本地保存，支持一键删除。
