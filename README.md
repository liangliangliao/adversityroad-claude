# 逆境之路 AdversityRoad

Unity 第三人称动作冒险 RPG：把现实中的困境转译为可战斗、可识别、可反击、可成长的敌人与场景。

> 看见弱点 → 面对困境 → 主动战斗 → 失败复盘 → 强化意志 → 再次挑战 → 逆袭成长

- 引擎：Unity 6000.5（URP，安卓/PC 双端）
- 打包：GitHub Actions + game-ci/unity-builder（推送到 `main` 或 `claude/**` 分支自动构建安卓 APK）

## 当前可玩内容（MVP 纵切片：训练武馆）

游戏启动后由 `GameBootstrap` 在运行时自动搭建整个场景（不依赖编辑器手工建场，保证 CI 无头打包可用）：

| 内容 | 说明 |
| --- | --- |
| 训练武馆场地 | 地面、围墙、训练柱，运行时烘焙 NavMesh |
| 玩家 | 移动/奔跑/跳跃/闪避（无敌帧）/轻重三段连招/格挡/精准格挡/锁定/内功 |
| 七维属性 | 生命、体力 + 意志/专注/自尊/边界/决断（心理属性归零触发心理硬直，不直接死亡） |
| 敌人 | 明日幻影（拖延轴）、咳声刺客（噪声轴）、自我怀疑低语（自我怀疑轴） |
| Boss | 拖延影魔：高血量高韧性，心理+物理混合攻击 |
| 机制物件 | 目标板（蓄力恢复心理属性）、拖延泥潭（减速+决断流失区域） |
| 技能 | 起步斩（高伤突进）、定心护体（消耗意志恢复心理属性） |
| 任务 | 3 条训练/主线任务，击败 Boss 通关；失败自动生成复盘任务（改进卡，不惩罚） |
| 胜负流程 | Boss 击败→胜利结算；玩家倒下→复盘面板→一键再战 |
| 心理安全 | SafetySettings：强度分级/主题禁用/台词柔化/恢复模式（全局心理伤害倍率） |

## 操作

### PC（编辑器 / 桌面构建）

| 按键 | 功能 |
| --- | --- |
| WASD / 鼠标 | 移动 / 镜头 |
| 鼠标左 / 右键 | 轻攻击 / 重攻击 |
| Space / Shift | 跳跃 / 闪避（流云步） |
| Ctrl（按下瞬间为精准格挡） | 格挡（边界盾）/ 定心格挡 |
| Q / R | 锁定敌人 / 内功（意志燃烧） |
| 1 / 2 | 技能：起步斩 / 定心护体 |
| E（靠近目标板按住） | 目标板蓄力，恢复心理属性 |

### 安卓真机

触屏操控层自动出现：左下虚拟摇杆移动，右半屏拖动转镜头，右下按钮组：
攻 / 重 / 闪 / 挡 / 锁 / 功 / 跳 / 技1 / 技2 / 互（目标板蓄力）。

## 获取安卓 APK

1. 推送到 `main` 或 `claude/**` 分支（或在 Actions 页手动 Run workflow）。
2. 打开仓库 **Actions → Build Unity Android → 对应运行**。
3. 页面底部 **Artifacts** 下载 `Android-Build`，解压得到 APK 安装到手机。

需要在仓库 Secrets 配置 `UNITY_LICENSE` / `UNITY_EMAIL` / `UNITY_PASSWORD`（已配置）。

## 项目结构

```
Assets/
├── Scenes/SampleScene.unity          # 构建入口场景（挂 GameBootstrap）
├── _Project/
│   ├── Materials/M_Base.mat          # URP Lit 基础材质（保证 Shader 进包）
│   ├── Scenes/SC_TrainingDojo.unity  # 编辑器一键建场产物（仅编辑器调试用）
│   ├── Editor/TrainingDojoBuilder.cs # 菜单：AdversityRoad → 一键搭建训练武馆
│   ├── Data/Json/                    # 场景/敌人/技能/问卷模板数据
│   └── Scripts/
│       ├── Core/          # GameBootstrap、GameManager、BattleFlow、GameEvents、SafetySettings
│       ├── Player/        # 移动、镜头、锁定、七维属性
│       ├── Combat/        # 连招、状态机、Hitbox/Hurtbox、心理伤害、技能、机制物件
│       ├── AI/            # 敌人 FSM（物理攻击 + 心理攻击）
│       ├── Personalization/ # 弱点轴、玩家画像、场景匹配、安全过滤
│       ├── Quest/         # 任务与复盘（改进卡）
│       ├── Mobile/        # 虚拟摇杆、触屏按钮、转镜头区
│       ├── UI/            # HUD 七维属性条、任务提示
│       └── Save/          # 本地 JSON 存档（数据最小化、可一键删除）
```

## 开发路线（对应产品方案）

- [x] 第 1-2 阶段：核心动作原型 + 战斗闭环（移动/连招/格挡/闪避/敌人 FSM/胜负）
- [x] 第 3 阶段：心理战系统（五维心理属性、心理攻击、定心格挡、心理硬直）
- [x] 训练武馆纵切片（含拖延影魔 Boss）+ 安卓触屏操控 + CI 自动打包
- [ ] 第 4 阶段：独居小屋（SC_HomeRoom）、噪声街区（SC_NoiseStreet）、求职荒原（SC_JobSquare）
- [ ] 第 5 阶段：个性化系统（问卷/文本导入 → 画像 → 场景与敌人匹配）
- [ ] 第 6 阶段：可发布 Demo（开场、3 个困境场景、2 个 Boss、技能树、设置菜单）

## 设计原则

- 游戏基于真实经历抽象为敌人原型与场景气氛，**不复刻真实人物/地点/事件**。
- 心理攻击只做成游戏机制（黑雾/低语/凝视/泥潭），**绝不输出可复制到现实的操控话术**。
- 失败不惩罚：死亡生成复盘任务，引导策略调整后再战。
- 隐私数据最小化：个人材料本地保存，支持一键删除。
