背包库（角色·装备库 →「背包」子菜单里选择背上/卸下）

用法：把下载好的背包模型直接丢进本目录即可，重启/重新打开面板后自动出现。
- 支持格式：.glb / .gltf / .fbx（.gltf 需连同 .bin 与贴图一起放入；.glb 自带贴图最省事）
- 也可直接丢 .zip 压缩包：编辑器会自动解压出模型（见 CharacterAssetZipExtractor）
- 文件名即背包名（显示在菜单按钮上）

生效规则：
- 背包由 BackpackRig 每帧从活动骨骼摆位：竖直跟随躯干（髋→颈）、朝向跟随角色面向，
  背板贴在背表面（上胸骨后方"躯干半厚+0.38 包厚"处）、包顶到肩线——任何模型/任何动作
  都竖立贴背、不穿身（只在装备瞬间摆一次姿态会把当时骨骼姿势烘进本地变换，是
  之前"背包横倒/沉进躯干"的根因）
- 朝向自动判定与作者无关：最薄轴=厚度、最大轴=高度；【肩带面=厚轴正侧】转向身体，
  模型自带的双肩带环正好套在双肩上
- 【肩带面反了怎么办】：个别模型肩带建在厚轴负侧时，放一个空标记文件即可翻转
  （同武器库 flipgrip 约定，一次配置永久生效）：
    · zip 解压出的背包：Resources/Characters/Backpacks/<背包名>/flippack.txt
    · 直接丢根目录的背包：Resources/Characters/Backpacks/flippack_<背包名>.txt
- 定尺：最大边≈0.26 身高
- 若模型无贴图呈白模，运行时会按贴图文件名自动接回 basecolor/normal（同武器库修复逻辑）
