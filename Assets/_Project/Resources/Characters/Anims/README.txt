把所有 Mixamo 动作 FBX 原样丢进这个目录，不用改名，例如：
  Maria WProp J J Ong@Idle.fbx
  Maria WProp J J Ong@Walking.fbx
  Maria WProp J J Ong@Running.fbx
  Maria WProp J J Ong@Great Sword Slash.fbx
  Maria WProp J J Ong@Side Kick.fbx  ... 等等
代码按文件名里的 @后缀（Idle / Side Kick / Great Sword Slash …）自动识别并分配到招式。
至少要有 Idle / Walking / Running 三个基础动作。放好后可删掉本 README。
