【武器库目录】每个模型文件 = 一件武器，文件名即游戏内武器名。
- 支持格式：.glb / .fbx / .gltf（.gltf 请连同 .bin 与贴图一起放入）
- 支持压缩包：把 .zip 丢进本目录，编辑器会自动解压出其中的模型/贴图并删除 zip
- 武器数量 = 本目录下模型文件数量，放入即出现在游戏内「角色」面板的武器库中
- 装备时自动隐藏角色自带兵器、挂到右手、按包围盒自动归一尺寸（最长边≈1.4m）
- 握向（哪端是柄）自动判定：优先看模型里的 grip/handle/hilt/tsuka 节点，其次看建模原点
  （艺术家通常把原点建在握柄处），再次看护手截面。绝大多数武器无需干预。
- 【握反了怎么办】：极少数完全对称、且导入时 pivot 被居中的武器（如某些蛇形刀）几何上
  无从判柄。此时在该武器目录下放一个空文件 flipgrip.txt 即可翻转握向（一次配置永久生效，
  随资源分发，无需改代码）：
    · zip 解压出的武器：Resources/Characters/Weapons/<武器名>/flipgrip.txt
    · 直接丢根目录的武器：Resources/Characters/Weapons/flipgrip_<武器名>.txt
- 若武器在手中朝向仍不对：在 Unity 中将该模型拖成 Prefab、调整旋转后放回本目录

【带剑鞘的成套武器】若模型里含 Scabbard/Sheath 节点（如 Highland 剑），游戏会识别为
可拔刀武器：默认收刀（整套挂左手横握鞘中部，剑身在剑鞘本地空间解析装配——剑尖抵鞘底、
剑柄自鞘口露出，每帧复位保证剑鞘永不分离），点 HUD 上「拔刀」键（或键盘 T）切换。
拔/收刀是两段式动画（与 Draw / Sheathing Sword 动作片段时长同步）：
收刀=先把剑移到鞘口、剑轴对准鞘轴，再沿鞘口缓缓插入；拔刀=先沿鞘轴抽出、再挥到右手。
